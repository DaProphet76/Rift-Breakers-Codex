# Rift-Breakers Codex

This repository contains the canonical world bible and narrative structure for the *Rift-Breakers Codex* universe.

## Repository Structure
- `/world_bible` — Immutable canon and metaphysical rules
- `/story` — Narrative content (POV-bound and unreliable)
- `/characters` — Character dossiers
- `/notes` — Non-canon ideas and drafts
